# PRODIGY_DS_Task5
Data analysis and visualization project using US accident dataset to explore patterns in weather, road conditions, and time, and identify accident-prone locations.


## 🎯 Objectives
- Analyze accident trends based on:
  - Weather conditions 🌧
  - Road conditions 🛣
  - Time of day ⏰  
- Identify accident hotspots 📍  
- Understand key contributing factors  
- Visualize patterns for better insights  


## 🛠️ Tools & Technologies
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Plotly / Folium (for maps, if used)  

## 📊 Analysis Performed
- Time-based analysis (hourly trends)
- Weather impact on accidents
- Road condition analysis
- Location-based accident clustering
- Hotspot visualization using heatmaps

## 📈 Visualizations
- Line charts (accidents over time)
- Bar charts (weather & road conditions)
- Heatmaps (accident-prone areas)
- Interactive maps (if implemented)
